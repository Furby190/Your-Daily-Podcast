# Your Daily Project
starter project webpack template

```git clone https://github.com/Furby190/Your-Daily-Podcast.git```

Go into directory

```cd Your-Daily-Podcas ```

### Assets

Install dependencies

```
npm install
```

Build assets:
```
npm run build
```

Watch assets for changes:
```
npm start
``` 

From now you can access project from url

http://localhost:3000
